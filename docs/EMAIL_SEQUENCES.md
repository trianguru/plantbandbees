# Email Automation Sequences - PlantBandBees

Complete email templates for nurturing waiting list leads from signup through Spring 2026 launch.

**Platform:** Mailchimp
**Total Sequences:** 2 (Welcome Series + Re-engagement)
**Total Emails:** 9
**Timeline:** 35 days + ongoing monthly updates

---

## Setup in Mailchimp

### Create Automation

1. Log in to Mailchimp
2. Go to **Automations**
3. Click **Create** → **Custom**
4. Automation name: **PlantBandBees Waiting List - Welcome Series**
5. Trigger: **When someone subscribes** to your list
6. Add tag: **waiting-list**

## Mailchimp Setup Instructions

### 1. Create Automation Workflow

1. **Automations** → **Create** → **Custom**
2. Name: "PlantBandBees - Welcome Series"
3. Trigger: **Subscribes to list**
4. Audience: Your waiting list
5. Save

### 2. Add Email Steps

For each email:
1. Click **+** to add an email
2. Choose **Email**
3. Set delay (immediate, 3 days, 7 days, etc.)
4. Design email:
   - Subject line
   - Preview text
   - From name: "PlantBandBees Team" or "Your Name"
   - Reply-to: Your email

### 3. Use Merge Tags

Personalize emails with merge tags:
- `*|FNAME|*` - First name
- `*|LNAME|*` - Last name
- `*|EMAIL|*` - Email address

Example: `Hi *|FNAME|*,`

### 4. Set Up Re-Engagement

1. Create separate automation
2. Trigger: **Has not opened email in 14 days**
3. Add Re-Engagement Email 1
4. Wait 7 days
5. Add Re-Engagement Email 2
6. Action: Tag as "inactive" or unsubscribe

### 5. Test Before Launching

1. Add your own email to test list
2. Run through entire sequence
3. Check formatting on:
   - Desktop email clients
   - Mobile Gmail
   - iPhone Mail
4. Verify all links work
5. Check merge tags populate correctly

## Email Best Practices

### Subject Lines
- Keep under 50 characters
- Use emojis sparingly (🌿 ✅ 🎉)
- Ask questions
- Create curiosity
- Be specific with value

### Preview Text
- 40-50 characters
- Complements subject line
- Teases content
- Don't repeat subject

### Body Content
- Conversational tone
- Short paragraphs (2-3 lines max)
- Use bullet points
- Bold key phrases
- Clear single CTA per email

### Send Times
Best days: Tuesday-Thursday
Best times: 10am, 2pm, 6pm Eastern

Test and adjust based on YOUR open rates.

### Mobile Optimization
- 60%+ will read on mobile
- Keep it scannable
- Large tap targets for links
- Single column layout

## Success Metrics

### Email Performance Targets

| Metric | Good | Great | Needs Work |
|--------|------|-------|------------|
| Open Rate | 25-35% | 35%+ | < 20% |
| Click Rate | 8-15% | 15%+ | < 5% |
| Reply Rate | 3-8% | 8%+ | < 2% |
| Unsubscribe | < 1% | < 0.5% | > 2% |

### Engagement Scoring

**Hot Leads (High Priority):**
- Opened 5+ emails
- Clicked 3+ links
- Replied to check-in email
- **Action:** Personal outreach, phone call

**Warm Leads:**
- Opened 3-4 emails
- Clicked 1-2 links
- **Action:** Continue sequence, add to Facebook retargeting

**Cold Leads:**
- Opened 0-2 emails
- No clicks
- **Action:** Re-engagement sequence, consider removing

## Optimization Tips

### A/B Test These Elements

1. **Subject lines**
   - Question vs. statement
   - Emoji vs. no emoji
   - Short vs. long

2. **Send times**
   - Morning vs. evening
   - Weekday vs. weekend

3. **Email length**
   - Short and punchy
   - Long and detailed

4. **CTA placement**
   - Top of email
   - Middle
   - Bottom

### Monthly Review

Check Mailchimp reports:
- Which emails have highest open rates?
- Which get most clicks?
- Which generate replies?
- Where do people unsubscribe?

Improve underperformers, replicate winners.
